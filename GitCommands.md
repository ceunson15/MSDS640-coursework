git init = used to create a repository for the terminal to access on local machine
git config = used to set the account username and email
git remote add origin = used to link to the GitHub repository we created
git status = shows what files have been added and which files are untracked
git add = prepares the file to get sent to the GitHub directory
git commit -m = creates a message that can be attached and read on the file
git push = sends the file to the GitHub for anyone to view and download
